DB central (db-central)

Esta es la base de datos MariaDB compartida. Hospeda dos schemas: uno para Passbolt y otro para el dashboard custom. Las filas que ves abajo son datos reales en vivo, consultadas en este momento.

Passbolt usa esta DB

Configurado en docker-compose con DATASOURCES_DEFAULT_HOST: db-central. Todas las contrasenas, usuarios y grupos viven aqui (cifrados con PGP).

Dashboard usa esta DB

Usuarios del panel, audit log y catalogo de objetivos monitoreados.

Checkmk NO usa esta DB

Checkmk guarda metricas en RRDtool (formato propietario de series de tiempo). No soporta MySQL externo por diseno — ver ARQUITECTURA.md.
